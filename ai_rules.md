# ai_rules.md — Правила для AI-ассистента

## Нельзя
- Перекодировать видео (только `-c copy`)
- Хардкодить токены и секреты
- Удалять логирование и обработку ошибок
- Менять схему БД без Alembic-миграции
- Смешивать Celery-задачи с Telegram-логикой

## Структура (не нарушать)
- `bot/`     — только Telegram: роутеры, middleware, FSM
- `core/`    — бизнес-логика, модели, сервисы
- `workers/` — только Celery-задачи
- `admin/`   — только FastAPI admin panel

## При изменениях
1. Создать тест
2. Обновить CHANGELOG_AI.md
3. Не трогать чужие модули без необходимости
