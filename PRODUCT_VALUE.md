# 🎯 MetaCleaner Bot — Продуктовая ценность

**Для кого:** Создатели контента, журналисты, активисты, обычные пользователи  
**Проблема:** Видео содержат метаданные (GPS, дата, устройство), которые могут раскрыть личную информацию  
**Решение:** Быстрое удаление метаданных без потери качества видео

---

## 💡 Зачем это нужно?

### Проблемы, которые решает MetaCleaner

1. **Приватность и безопасность**
   - Видео с телефона содержит GPS координаты места съемки
   - Метаданные раскрывают модель устройства и время съемки
   - Это может быть опасно для журналистов, активистов, обычных людей

2. **Профессиональный контент**
   - Удаление служебной информации перед публикацией
   - Очистка метаданных для клиентов
   - Подготовка видео для соцсетей

3. **Простота использования**
   - Не нужно устанавливать программы
   - Работает прямо в Telegram
   - Обработка за 1-2 минуты
