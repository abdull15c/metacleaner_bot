# MetaCleaner Bot — Project Manifest

**Версия:** 1.0.0
**Статус:** MVP, локальный запуск

## Стек
- Python 3.11, aiogram 3.7, FastAPI, Celery, Redis, SQLite, FFmpeg

## Что делает
- Принимает видеофайлы и YouTube-ссылки (с подтверждением прав)
- Удаляет метаданные через `ffmpeg -c copy -map_metadata -1`
- Автоудаляет temp файлы после отправки
- Веб-панель с задачами, пользователями, рассылками, настройками

## Очереди Celery
- `video`     — обработка видео и отправка (2 воркера)
- `cleanup`   — удаление файлов (1 воркер)
- `broadcast` — рассылки, изолированно (1 воркер)
